## Vim Awesomness

After cloning this repo...

    ln -s ~/.vim/vimrc ~/.vimrc
	
Checkout the vundle submodule
    
    git submodule update --checkout
   
Install vim plugins
   
    vim +PluginInstall +qall



#### Notes

`<ESC>` has been remapped to `jk`

Leader key has been mapped to ","
